# Charan Teja – Developer Portfolio

A modern, animated developer portfolio built with Next.js 15, TypeScript, Tailwind CSS v4, Framer Motion, and Lenis smooth scrolling.

## Features

- **Glassmorphism Navbar** – Fixed pill-shaped nav with scroll-aware blur effect
- **Hero Section** – Full-viewport with floating tech icons and parallax animations
- **Selected Works** – Sticky-stack scroll effect showcasing 5 projects
- **Tech Arsenal** – Three-column skill grid (Frontend / Backend / Tools) with level badges
- **Pixels at Play** – Creative lab bento grid with hover effects
- **Smooth Scrolling** – Lenis v1 integration via `requestAnimationFrame`
- **Dark Theme** – Deep `#030303` background with emerald/blue accent palette

## Stack

- **Framework**: Next.js 15 (App Router, Turbopack)
- **Language**: TypeScript
- **Styling**: Tailwind CSS v4
- **Animation**: Framer Motion
- **Scroll**: Lenis 1.x
- **Font**: System font stack (`-apple-system`, `Segoe UI`, Roboto, …)

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the portfolio.

## Build

```bash
npm run build
```
